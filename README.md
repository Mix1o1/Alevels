# A-Level Topical Papers

This repository contains topical summaries and questions for A-Level subjects from Cambridge International Examinations (CIE).

## Subjects Included

- **Physics** - Papers 1, 2, 4
- **Chemistry** - Papers 1, 2, 4
- **Mathematics** - Papers 1, 3

## Structure

Each subject is organized by paper, with topics listed for each paper. Relevant examination questions from 2025 papers are included for practice.

## How to Use

1. Navigate to the subject folder (Physics, Chemistry, or Mathematics)
2. Select the specific paper
3. Review the topics listed
4. Practice with the included questions from 2025 papers
